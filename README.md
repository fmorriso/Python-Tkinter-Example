# Python Tkinter Example
A simple Python program that roughly follows the Tkinter Tutorial with some
modifications for window scaling using PyAutoGUI.

## Tools Used

| Tool        |  Version |
|:------------|---------:|
| Python      |   3.13.0 |
| Tkinter     |      8.6 |
| PyAutoGUI   |   0.9.54 |
| VSCode      |   1.95.1 |
| PyCharm     | 2024.2.4 |
